# Changelog 索引

本目录保存项目变更记录，采用“一个版本一个文件”的方式维护。

## 文件约定

- `Unreleased.md`：当前开发中变更（待发布）。
- `vX.Y.Z.md`：已发布版本变更（例如 `v0.0.5.md`）。

## 维护约定

- 分类保持：`新增` / `修复` / `变更` / `文档`。
- 发布时将 `Unreleased.md` 内容迁移到新版本文件，并重建空模板。
- 历史版本文件只做勘误，不改写语义与顺序。
